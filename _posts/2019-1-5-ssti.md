# What is SSTI
`SSTI` Server Side Template Injection , one of the injection attack . the `SSTI` sometime can getshell !
```php
$output=$twig->render('Dear '{first_name},array("first_name" => $user.firstname));
```
```javascript
Dear {$user.first_name}
```
